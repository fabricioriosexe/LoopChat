LoopChat es una aplicación de mensajería instantánea diseñada para ofrecer una experiencia de comunicación en tiempo real, rápida y segura. 
La plataforma permite a los usuarios conectarse con sus amigos, familiares y colegas a través de mensajes de texto, multimedia, chats grupales, y llamadas de voz
y video, todo mientras se garantiza la privacidad mediante cifrado de extremo a extremo. LoopChat es altamente escalable, integrando MongoDB y MySQL como sistemas de
almacenamiento, asegurando que tanto los mensajes como los datos de usuario se gestionen de manera eficiente y segura.

Funcionalidades principales:
Mensajería en tiempo real: Comunicación instantánea mediante WebSockets.
Chats individuales y grupales: Capacidad para enviar mensajes a una sola persona o grupos de hasta 100 personas.
Mensajería multimedia: Compartir imágenes, videos, documentos, y mensajes de audio.
Notificaciones en tiempo real: Informar a los usuarios cuando reciben mensajes nuevos.
Cifrado de extremo a extremo: Protección de la privacidad de las conversaciones.
Búsqueda avanzada: Permite buscar mensajes pasados o archivos en el historial de chat.
Estado de usuario: Mostrar el estado de conexión (en línea, desconectado, escribiendo).
Llamadas de voz y video: Comunicación a través de llamadas de audio y video.
